Nom!Nom!

立命館大学OICキャンパス キッチンカー評価プラットフォーム

学生による学生のための、キャンパス内キッチンカー情報共有サービス

プロジェクト開始: 2025年5月 | ベータ版予定: 2025年10月 | 正式リリース: 2026年3月

プロジェクト一覧

プロジェクト名 概要 ステータス 進捗率 予定リリース
Nom!Nom! Web版 メインのWebアプリケーション 開発中 95% 2025年10月
Nom!Nom! Mobile App iOS/Android対応モバイルアプリ 開発中 40% 2026年1月
管理者向けダッシュボード キッチンカー事業者向け管理画面 計画中 5% 2026年4月
Nom!Nom! 全国展開版 他大学キャンパス対応版 構想中 0% 未定

Nom!Nom! 主要機能

🍽️ キッチンカー検索・閲覧機能

  • リアルタイムでのキッチンカー位置情報表示
  • 営業時間・メニュー情報の確認
  • カテゴリ別検索（和食・洋食・スイーツ等）
  • 価格帯・評価による絞り込み検索

⭐ レビュー・評価システム

  • 5段階評価による総合評価
  • 味・コスパ・接客の個別評価
  • 写真付きレビューの投稿
  • 学生認証によるレビューの信頼性確保

👥 コミュニティ機能

  • お気に入りキッチンカーの登録
  • 新着・営業開始の通知機能
  • 友達とのおすすめ共有機能
  • 学年・学部別のランキング表示

📱 モバイル対応（予定）

  • ネイティブアプリでの快適な操作
  • プッシュ通知によるお知らせ
  • オフライン閲覧対応
  • GPS連動による現在地からの検索

使用技術・開発環境

フロントエンド:
  • React.js 18 + TypeScript
  • Next.js 14 (App Router)
  • Tailwind CSS
  • Framer Motion (アニメーション)
バックエンド・インフラ:
  • Vercel (Webアプリデプロイ)
  • Cloudflare (CDN・セキュリティ)
  • Firebase (認証・データベース)
  • GitHub Actions (CI/CD)
モバイルアプリ:
  • Flutter (クロスプラットフォーム開発)
  • Dart (プログラミング言語)
  • Firebase Integration
  • iOS/Android対応
その他・コミュニティ:
  • Minecraftサーバー運用
  • Discord (チーム連携)
  • ゲーム開発・MOD制作
  • AI活用ツール
